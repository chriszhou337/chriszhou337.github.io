Cities Skylines Modding

Essential mods for Cities Skylines and Cities Skylines 2

Notice: Modding is only supported for PC

Recommended Mods for Cities Skylines 1

  • 81 Tiles: Allows for the unlocking of all 81 tiles on the map.
  • Network Anarchy: Allows placement of roads and other objects without restrictions.
  • Move It: Allows movement and rotation of objects freely.
  • Intersections Marking Tool: Allows for road and intersection detailing
  • TM:PE (Traffic Manager: President Edition): Manage traffic flow and customize road behavior.
  • Find It! 2: Search for assets and mods in game with CTRL + F.
  • Loading Screen Mod Revisited: Boosts game loading times and diagnoses crashes.
  • Harmony: Mod dependancy.

Recommended Mods for Cities Skylines 2

  • Advanced Road Tools: Provides tools to control roads and their behaviour.
  • Traffic: Manage traffic and lane management.
  • Move It: Allows movement and rotation of objects freely.
  • Anarchy: Disables error checking for tools to place roads and buildings freely
  • 529 Tiles: Unlocks all tiles on map for gameplay
  • Better Bulldozer: Easier deletion of structures
